## 📌 Overview
Academic pressure, sleep deprivation, and environmental shifts significantly impact students' mental health. This project leverages **Machine Learning (ML)** to predict student stress levels and identify the primary triggers behind academic burnout. By analyzing key behavioral and physiological data, the system provides early insights to help students and institutions take proactive wellness measures.

## 📊 Key Features
* **Predictive Modeling:** Utilizes algorithms like Random Forest, XGBoost, and Logistic Regression to classify stress levels (Low, Medium, High).
* **Feature Importance Analysis:** Identifies which factors (e.g., sleep quality, study hours, peer pressure) contribute most to high stress.
* **Interactive Dashboard (Optional):** A user-friendly interface built with Streamlit/Flask for real-time stress assessment.
* **Data-Driven Insights:** Provides actionable recommendations based on the predicted stress risk.

## 🧬 Data Features Analyzed
The model evaluates various metrics, including:
* **Academic:** GPA, study hours, exam pressure, extracurricular load.
* **Physiological:** Sleep quality, physical activity, heart rate variability (if applicable).
* **Social/Environmental:** Peer pressure, financial anxiety, family support.

* **Libraries:** Pandas, NumPy, Scikit-Learn, XGBoost, Matplotlib, Seaborn
